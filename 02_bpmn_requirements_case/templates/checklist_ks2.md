# Чек-лист акта КС-2 для Розенберг (метро "Юго-Западная")

- [ ] Номер и дата акта соответствуют журналу учёта
- [ ] Объект: "Строительство станции метро Юго-Западная, этап 2"
- [ ] Позиции сметы соответствуют проектной документации (шифры ТЕР/ФЕР)
- [ ] Арифметика проверена, НДС выделен
- [ ] Акт подписан прорабом и мастером
- [ ] В Битрикс24 приложены:
  - [ ] Журнал выполненных работ (форма КС-6а)
  - [ ] Накладные на материалы
  - [ ] Акт скрытых работ (если требуется)
- [ ] Для субподрядчика: указаны его работы, приложен его акт
- [ ] Поле "Комментарий для ПТО" заполнено (особенности)
- [ ] Если акт идёт в Росстройконтроль — выбрана соответствующая галочка
